# 🐾 Week04: Animals / 动物

## 🌟 Sentence Pattern / 句型模式

**I like [animal]. / 我喜欢[animal]。**

---

## 📚 Vocabulary / 单词列表

- cat - 猫  
- dog - 狗  
- elephant - 大象  
- lion - 狮子  
- rabbit - 兔子  
- tiger - 老虎
- monkey - 猴子
- bear - 熊
- panda - 熊猫
- bird - 鸟

---

## 📝 Example Sentences / 示例句子

- I like cat（猫）。  
- I like dog（狗）。  
- I like elephant（大象）。  

---

## 🎲 Game Idea / 游戏互动

- 动物叫声模仿游戏：学动物叫声并用英语说名字。  
- 画动物游戏：让孩子画喜欢的动物并说单词。  
- 动物分类游戏：按大小或种类分组动物图片。
